# Simple Landing Webpack

Just another Webpack boilerplate

## Features

- Webpack 5
- Babel
- SASS
- ESLint
- Prettier

## Installation

Clone repository

```bash
git clone https://github.com/rafaelszago/simple-landing-webpack
```

Install dependencies

```bash
yarn install
```

## Development

Start server in development mode

```bash
yarn start
```

## Build

Build files to production

```bash
yarn build
```

## Lint

Lint your code

```bash
yarn lint
```
